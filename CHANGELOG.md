## Release Notes

### Version 0.1.0 - Initial Release

This is the first public release of the Mekaneck framework. It includes:

1. **Complete Python validation package** proving all theoretical claims
2. **Comprehensive documentation** of the pharmaceutical Maxwell demon concept
3. **Professional development setup** ready for GitHub publication
4. **Docker containerization** for reproducible validation
5. **10 independent validators** with quantitative metrics

**This release focuses on validation.** The Rust core implementation will follow in Q1 2025.

### Breaking Changes
- None (initial release)

### Deprecations
- None (initial release)

- Harmonic network construction can be slow for N_max > 150 (addressed in Rust implementation)
